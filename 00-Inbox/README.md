# Inbox

Drop any raw ideas, quick notes, links, or thoughts here. Process regularly by moving notes to the appropriate folder and adding proper tags and links.

## Quick Capture Tips
- Don't overthink it — just get it out of your head
- Add a one-line summary so future-you knows what it's about
- Process this folder weekly: refine, link, and relocate notes
